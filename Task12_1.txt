1.Создать свой репозиторий в таким же именем, как и имя пользователя

	git init

2. Склонировать его на свой компьютер в отдельную папку

	git clone git@github.com:AndreiPamakha/AndreiPamakha.git

3.Склонировать себе следующий репозиторий в отдельную папку https://github.com/testrusau/testrusau

	git clone git@github.com:testrusau/testrusau.git


4.Выберите один из любых вариантов и выполните его.

	5. Вариант 1: Скопировать данные из склонированного репозитория и вставить их в ваш репозиторий из шага 2

		cd testrusau
		cp -r testrusau AndreiPamakha

	6. Вариант 2 (задание со звездочкой): запушить данные из репозитория artichokeee в ваш репозиторий из шага 2

7. Открыть файл README.md и поочередно заменить каждый блок на вашу информацию . Каждое изменений блока делайте через отдельный коммит.
